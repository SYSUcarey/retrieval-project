# 搜索引擎初探

## 项目概览
基于课程所学内容，实现一个简单的搜索引擎功能，实现以下功能（不要求可视化界面，仅完成控制台输出）：

1. 拼写检查（参考最小编辑距离原理）
2. 倒排索引
3. 使用TF/IDF进行文档排序

***

## 项目环境
- java 9.0.1
- idea项目管理

***

## 项目目录
- data
  - page：搜索引擎的数据文档语料，来源于百度词条
- lib：使用到的jar包，具体包括：lucene 和 commons-lang等
- src：项目源码
- assets：图片等资源
